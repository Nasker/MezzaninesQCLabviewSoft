Agilent 81150 Instrument Driver Readme

To rate this instrument driver, go to the driver download page on the Instrument Driver Network.

1. Overview

Instrument Driver Technology: LabVIEW Plug and Play (project-style)
Manufacturer: Agilent Technologies
Supported Language(s): LabVIEW
Supported Model(s): 81150A, 81160A
Model(s) Tested: 81150A
Interface(s): GPIB, USB, Ethernet
Firmware Revision(s) Tested: 81150A: 1.0.0.6-4, 2.0.9.0-4.4
81160A: 1.0.3.0-2.6


Certified: Yes
NI Supported: Yes
Source Code Available: Yes

Driver Revision: 1.2
Original Release Date: 9/4/2008
Current Revision Date: 01/17/2013

2. Required Software

Some software components need to be installed before using this instrument driver. The minimum versions of these components are listed below, and can be downloaded from the Download Site.

VISA 4.0 or later
LabVIEW 8.2 or later

Refer to the LabVIEW Help for more information about software requirements. You access the LabVIEW Help by selecting Help»Search the LabVIEW Help.

3. Known Issues

Pattern related commands are not supported right now. To report issues or provide feedback about this instrument driver, please send an email to instrument.drivers@ni.com.
